# Life-Hedge Simulator → 보험사 바이럴 마케팅 도구 피벗 전략

> **컨셉**: MBTI처럼 가볍게 시작 → 실질 데이터로 인사이트 → 자연스러운 보험 노출

---

## 1. 핵심 차별화: "가벼운 톤 + 진짜 데이터"

### ❌ 기존 보험 마케팅의 문제
- 공포 마케팅: "암에 걸리면 큰일납니다" 😱
- 무거운 상담: "설계사 만나세요"
- 결과: 거부감, 공유 안 함

### ✅ 우리의 접근
- MBTI처럼 가벼운 진입: "내 방어 유형은?" 🎮
- 하지만 실질 데이터: 확률 %, 치료비 원
- 결과: 재미있어서 공유, 데이터라서 신뢰

## 2. 건강 방어 유형 4종

| 유형 | 이름 | 특징 | 상태 |
|------|------|------|------|

| ⚖️ **PREP** | 준비된 현실주의자 | 어느 정도 대비됨 | 괜찮은 편 |
| � **HOPE** | 긍정 에너지 | 부족하지만 괜찮겠지 | 점검 필요 |
| 🎲 **YOLO** | 오늘을 사는 자 | 아직 준비 안 됨 | 시작해볼까? |

## 4. 바이럴 포인트

### 🔥 공유하고 싶은 이유
1. **유형 비교**: "너 뭐 유형이야? 나 PREP ㅋㅋ"
2. **숫자 충격**: "헐 58세에 12%래 ㅋㅋ"
3. **방어력 비교**: "나 42%밖에 안 돼 ㅠ"

### 💬 예상 반응
- "오 이거 재밌다 해봐"
- "근데 데이터 어디서 나온 거야?" → HIRA 출처
- "나도 해봐야겠다" → 바이럴 확산

## 5. MVP (최소 기능 제품) 구성

### Phase 1: 바이럴 테스트 (1주)
- 3개 질문 → 결과 카드 생성
- 카카오톡 공유 버튼
- 결과 이미지 저장

### Phase 2: 리드 수집 (2주)
- "상세 분석 보기" → 이메일 입력
- 보험 상담 신청 CTA

### Phase 3: B2B 상품화
- 화이트라벨 (로고/색상 커스텀)
- 보험사 CRM 연동
