# Life-Hedge Simulator 개선안

## 현재 문제점 분석

### 1. 위험 구간 임계값 (2%) 의 근거 부족

**현재 상태:** 발병 확률 2% 이상을 "위험 구간"으로 표시

**문제점:** 
- 2%라는 숫자에 대한 근거가 없음
- 사용자가 "왜 2%가 위험한 건지" 이해하기 어려움

**개선 방향:**
1. **상대적 위험도 표시**: 절대 확률보다는 "평균 대비 몇 배 위험" 형식으로 표시
2. **사용자 정의 임계값**: 슬라이더로 위험 임계값을 직접 조절 가능하게
3. **위험도 설명 추가**:
   - 1% 미만: 💚 안전 (100명 중 1명 미만)
   - 1~3%: 🟡 주의 (100명 중 1~3명)
   - 3~5%: 🟠 위험 (20~33명 중 1명)
   - 5% 이상: 🔴 고위험 (20명 중 1명 이상)

> **참고**: 일반적으로 의료계에서는 5% 이상을 "높은 위험"으로, 1% 미만을 "낮은 위험"으로 분류하는 경우가 많습니다.

---

### 2. 보험의 가치가 과소평가됨

**현재 상태:** 투자 수익률만 반영 → 투자가 항상 유리해 보임

**실제 현실:**
- 투자는 **손실 가능성**이 있음 (원금 손실)
- 보험은 **확정 금액**을 지급함 (리스크 헷지)
- 질병 발생 시점을 예측할 수 없음 (투자 기간 불확실)

**개선 방향:**

#### A. 투자 리스크 반영
```
시나리오 1: 낙관적 (7% 수익률)
시나리오 2: 평균적 (5% 수익률)  
시나리오 3: 비관적 (2% 수익률 또는 원금 손실)
```

#### B. 조기 발병 시나리오 추가
- "만약 45세에 암이 발생하면?" → 투자 기간이 짧아 보험이 유리
- "만약 75세에 발생하면?" → 투자 기간이 길어 투자가 유리

#### C. 보험의 숨겨진 가치 표시
- **진단금 즉시 지급**: 투자는 환매 시간 필요
- **세금 혜택**: 보험료 세액공제 (연 100만원 한도)
- **심리적 안정감**: 정량화 어렵지만 실제 가치 있음

---

### 3. 투자 수익률의 비현실성

**현재 상태:** 고정 수익률 5%로 계산

**문제점:**
- 실제 주식시장 변동성 무시
- 복리 효과가 과장됨 (중간에 손실 없다는 가정)

**개선 방향:**
1. **변동성 시뮬레이션**: 몬테카를로 시뮬레이션으로 최악/최선 시나리오 표시
2. **실질 수익률**: 물가 상승률(3%) 차감한 실질 수익률 표시
3. **시장 리스크 경고**: "투자는 원금 손실 가능" 명시

## 구체적 UI/UX 개선안

### 개선 1: 위험도 설명 패널 추가
```
┌─────────────────────────────────────┐
│ 📊 발병 확률 해석 가이드           │
├─────────────────────────────────────┤
│ 💚 1% 미만  → 100명 중 1명 미만    │
│ 🟡 1~3%    → 100명 중 1~3명       │
│ 🟠 3~5%    → 30명 중 1명 수준     │
│ 🔴 5% 이상 → 20명 중 1명 이상     │
└─────────────────────────────────────┘
```

### 개선 2: 시나리오 비교 차트
현재: 투자 단일 라인
개선: 3개 라인 (낙관/평균/비관) + 보험 라인

```
         │
   3억   │         ▲ 투자 (낙관 7%)
         │       ▲   ▲ 투자 (평균 5%)
   2억   │     ▲       ▲ 투자 (비관 2%)
         │   ▲ ━━━━━━━━━━ 보험 (확정)
   1억   │ ▲
         └──────────────────────────
           40세  50세  60세  70세
```

### 개선 3: "조기 발병" 시나리오 버튼
"만약 50세에 발병하면?" 버튼 클릭 시:
- 해당 시점까지의 투자 누적액 vs 보험금 비교
- 보험이 유리한 케이스 부각

## 결론

현재 시뮬레이터는 **투자 수익률을 과대평가**하고 **보험의 리스크 헷지 가치를 과소평가**하고 있습니다.

핵심 개선점:
1. **"언제 아플지 모른다"** → 조기 발병 시나리오 추가
2. **"투자는 항상 오르지 않는다"** → 비관적 시나리오 추가
3. **"확률 숫자의 의미"** → 해석 가이드 추가

이 개선을 통해 사용자가 보다 **균형 잡힌 시각**으로 보험과 투자를 비교할 수 있습니다.
